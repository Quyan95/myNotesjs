Массивы, объекты	                Ссылаются (по ссылке)

null - переменная пустая

--------------Массивы------------- let arr = []  
[n, m] - элементы массива  
--------------------------  
.shift() - забирает(!не копия!) первый элемент массива  
.pop() - забирает(!не копия!) последний элемент массива  
.unshift(a) - добавляет a элемент в начало массива  
.push(a) - добавляет a элемент в конец массива  
b.concat(a) - соединяет массив b к массиву a  
.indexOf(a) - показывает индекс a элемента в массиве  
.join() - превращает массив в строку  
.split() - превращает строку в массив и делает одним элементом массива. Если  .split('') - cоздает несколько элементов в массиве.  
.splice(n, m) - удаляет с массива начиная с индекса n в количестве m. Можно указывать отрицательные цифры в индексе n и удаляется с конца массива.  
.includes(a) - ищет есть ли a элемент в массиве и возвращает boolean(true(есть) или false(нет))  
.reverse() - возвращает массив в обратном порядке.  
.startsWith("He") - начинается ли начало строки внутри "Hello World". Это boolean возвращает true(да) или false(нет)  
.endsWith("rld") - начинается ли начало строки внутри "Hello World". Это boolean возвращает true(да) или false(нет)  
.fill(n, m, o) - переписывает массив заполняя все на n, начиная с m до o  
.entries() - возвращает массивы (пример в main.js 418)  
.keys() - возвращает индексы (пример в main.js 418)  
.values() - возвращает значения (пример в main.js 418)

--------------Циклы---------------(работают только с массивами.)  
.find(function(a)...) - ищет тот элемент который нужен  
.forEach(function(a)...)  - проходится по всем элементам и нельзя прервать  
.findIndex(function(a)...) - ищет индекс элемента  
.filter(function(a)...) - отбирает подходящие элементы(работает как сито). !создает новый массив  
.map(function(a)...) - проходится по всем элементам и преображает(умножить, поделить, округлить). !создает новый массив  
.reduce(function(acc, num)..., start) - является аккамулятором. Делает все вышеперечисленное. start(нумерация) это начало аккума. Лучше загуглить  
.sort(fuction(a,b)...) занимается сортировкой
